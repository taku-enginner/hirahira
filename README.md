# ひらひら落ち葉

このシェルスクリプトは、ターミナル上で「ひらひら落ち葉」をアニメーション表示するミニアプリです。RUNTEQミニアプリWeekの課題作品です。

## 実行方法
1. リポジトリをクローンします
```sh
git clone git@github.com:taku-enginner/hirahira.git
```
2. ローカルリポジトリで実行します

```sh
./hirahira.sh
```

## 注意点

- ターミナルのサイズが小さいと、画面崩れが起こる場合があります。
- 強制終了（Ctrl + c）した場合は、`tput cnorm` を実行してカーソルを元に戻してください。

## 機能

- ターミナル中央にタイトルを表示
- 木と根を描画
- 落ち葉がひらひらと落ちるアニメーション
- 終了時にメッセージとカウントダウン表示
